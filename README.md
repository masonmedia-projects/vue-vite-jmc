## Intro

Repository for redesign of official website for Jen Mason Consulting. Tech stack includes Vue3, Vite, Bootstrap 5, and Cockpit CMS.

## Details

Clone repo
`npm install`

### Build 

`npm run build`

### Manual Deploy

Zip `/dist` folder
Deploy contents to `public_html`
